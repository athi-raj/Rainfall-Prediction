## Overview
This project analyzes daily rainfall data from different states across India. The dataset is cleaned and visualized to understand rainfall patterns, trends, and variations. The project demonstrates data preprocessing, exploratory data analysis (EDA), and visualization using Python.

## Dataset
The dataset contains state-level rainfall information with the following features:

- id
- date
- state_code
- state_name
- actual (Actual Rainfall)
- rfs (Forecast Rainfall)
- normal (Normal Rainfall)
- deviation

## Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Tasks Performed
- Loaded and inspected the dataset
- Checked and handled missing values
- Removed duplicate records
- Converted date columns to datetime format
- Created monthly rainfall trends
- Visualized rainfall distribution
- Performed state-wise rainfall analysis
- Detected outliers using boxplots
- Generated correlation heatmaps
